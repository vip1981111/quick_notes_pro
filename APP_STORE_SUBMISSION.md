## 11. App Store Submission Checklist

### Pre-Submission

- [ ] IPA file ready (21MB - Jan 12, 2026)
- [ ] Bundle ID matches App Store Connect
- [ ] Version 1.0.0 (Build 1)
- [ ] Privacy Policy URL live and accessible
- [ ] Terms of Service URL live and accessible
- [ ] Support email configured

### Transporter Upload

- [ ] Download Transporter from Mac App Store
- [ ] Sign in with Apple ID (Developer account)
- [ ] Drag & drop IPA file
- [ ] Click "Deliver"
- [ ] Wait for processing (5-15 minutes)
- [ ] Check for errors in App Store Connect

### App Store Connect Setup

- [ ] Create new app in App Store Connect
- [ ] Select Bundle ID: com.mohammedabdullah.quicknotespro
- [ ] Add English (Primary) localization
- [ ] Add Arabic localization
- [ ] Fill in app description (EN & AR)
- [ ] Add keywords (EN & AR)
- [ ] Add subtitle (EN & AR)
- [ ] Set support URL
- [ ] Set Privacy Policy URL
- [ ] Upload screenshots (all required sizes)
- [ ] Set age rating (4+)
- [ ] Select categories (Productivity, Utilities)

### In-App Purchases Setup

- [ ] Go to Features > In-App Purchases
- [ ] Create Subscription Group: "Quick Notes Pro Premium"

  - Price: $1.99
  - Duration: 1 Month

  - Display Name (AR): بريميوم شهري

## 12. Common Rejection Reasons (Avoid These)

1. **Broken Privacy Policy link** - Test URL before submission
2. **Missing subscription info** - Add all required metadata
3. **Screenshots don't match app** - Use real app screenshots
4. **Incomplete metadata** - Fill all required fields
5. **Crash on launch** - Test thoroughly before submission
6. **Placeholder text** - Remove all "Lorem ipsum" text
7. **Missing restore purchases** - Must have restore button
